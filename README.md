# Developer Profile Generator
An application that creates a developer profile based on input by the user.

## Table of Contents 

* [Description](#Description)
* [User Story](#UserStory)
* [License](#License)
* [Badges](#Badges)

## Description
This is a command-line application that dynamically generates a PDF profile from a GitHub username. The application will be invoked with the following command:

```sh
node index.js
```

The user will be prompted for a favorite color, which will be used as the background color for cards.

Click [here](https://ofagbure/github.io/Developer-Profile-Generator/) to see the deployed site!

## User Story
AS A project manager

I WANT a developer profile generator

SO THAT I can easily pull together the profiles of code collaborators when preparing reports
